1. Export your JIRA issues as CSV (include: Summary, Issue key, Issue Type, Status, Parent key, Parent summary, Time Spent)
2. Click "Import CSV File" and select your file
3. Expand user stories to see tasks grouped by status
4. Export the organized data back to Excel if needed
